回路セットアップ

ゼロ知識証明の生成に必要な回路アーティファクトのダウンロードと設定方法を説明します。

回路アーティファクトが必要な理由

回路アーティファクトは、引き出し（テレポート）に必要なゼロ知識証明を生成するために不可欠です。これらのアーティファクトには、信頼されたセットアップの結果と証明生成に必要なパラメータが含まれています。

公式マニフェストダイジェスト

ダウンロードしたアーティファクトの整合性を検証するために、公式のマニフェストダイジェストが提供されています。

インストール

cargo install --path ../circuit-setup

設定

回路セットアップツールは環境変数で設定できます。デフォルトのアーティファクト保存先ディレクトリは自動的に作成されます。

コマンド

ダウンロード

回路アーティファクトをダウンロードします:

zerc20-circuit-setup download --version 1.1.0

このコマンドは必要なすべてのアーティファクトをダウンロードし、マニフェストダイジェストに対して整合性を検証します。

テスト

ダウンロードしたアーティファクトで証明の生成/検証をテストします:

zerc20-circuit-setup test
